# CapitalGuard Trader

**Disciplined Single-Name Trading • Capital Protection First**

A transparent educational project showing how to grow a small account using **one ticker at a time**, strict risk management, and consistent daily execution.

### Goal
Start with **$100** and build consistent profits through small, high-probability trades — with the long-term aim of scaling into meaningful daily income.

> **Important**: This is an educational project. Trading involves substantial risk of loss. Past performance is not indicative of future results.

---

### Core Philosophy

- **Protect Capital First**
- One ticker at a time (high focus, deep understanding)
- **Mandatory trailing stop-loss** on every trade
- No fixed profit targets — guidance levels only (let winners run when strong)
- Small position sizing with maximum 0.75% risk per trade
- Trade only during regular market hours

---

### Trading Rules

| Rule                    | Details                                      |
|-------------------------|----------------------------------------------|
| **Tickers**             | One primary ticker at a time                 |
| **Position Size**       | Max 0.75% account risk per trade             |
| **Stop Loss**           | **Mandatory** — ATR-based trailing stop      |
| **Profit Targets**      | Guidance only: T1 (+1.2%), T2 (+2.5%), T3 (+4%+) |
| **Daily Loss Limit**    | Stop trading if -2% on the day               |
| **Strategy**            | Trend-following pullback entries             |
| **Timeframe**           | Intraday to swing (primarily daily setups)   |
